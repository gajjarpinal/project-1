# Interactive Personal Data Collector

A simple Python program that collects personal information from the user, displays the data type and memory address of each variable, and calculates the approximate birth year based on the entered age.

## Features

- Collects user information:
  - Name
  - Age
  - Height
  - Favorite Number
- Displays:
  - Variable values
  - Data types
  - Memory addresses using `id()`
- Calculates approximate birth year
- Uses Python's `datetime` module

## Python Concepts Used

- Variables
- Input and Output
- Type Casting
- f-Strings
- `type()` function
- `id()` function
- `datetime` module
